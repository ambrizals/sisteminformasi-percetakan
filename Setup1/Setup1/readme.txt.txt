Selamat datang di Sistem Informasi Percetakan

Sistem Informasi Percetakan merupakan salah satu aplikasi sistem informasi yang berguna untuk mempermudah memperoleh informasi mengenai karyawan, daftar bahan, dan pemesanan.

Sebelum menginstall aplikasi ini harap persiapkan :
1. MySQL Server / MySQL Community Edition / XAMPP (Untuk Server)
2. Net Framework 4.0 (Jika tidak terinstall)

Deskripsi Aplikasi :
1. Percetakan Client
Aplikasi ini digunakan untuk client yang digunakan untuk mengelola seluruh sistem.

2. Percetakan Server
Aplikasi ini digunakan untuk server yang digunakan untuk mengelola database baik itu backup atau restore dan pemasangan awal database
